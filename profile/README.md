# Welcom to DSRemapper

![DSRBanner](../Images/DSRBanner.png)

DSRemapper stands for (Device Switcher and Remaper) and it's a modular app that can remap almost any input device in a computer into an output device (usually virtual, but can be physical).

This remap occurs inside a Remapper plugin, which can be simple or complex, giving the user greate flexibility to reasign it's devices in any way.
